第一周作业

题目

  1. 请在可见，红外，微波和射频各选择一个合适单位的单色电磁波，计算对应的波长（nm或μm），波数（\(\mathrm{cm^{-1}}\)），频率（\(\mathrm{Hz}\)），能量（\(\mathrm{eV}\)和J）。

  2. 把6000K的太阳，300K的地球和2.7K的宇宙分别作为一个黑体。 1）画出谱能量密度的频率分布图 2）找到最大辐射对应的频率以及该频率对应的波长、能量密度和电场强度。

  3. 考虑一束功率4W输出的均匀圆形激光打在一个电子能级近似两能级的原子体系，光斑直径为1mm，原子体系的跃迁偶极大小为1 Debye (1 D = 3.34×10⁻³⁰ C·m)。 1）请计算电场强度和拉比振荡频率。 2）实验室常用的能观察到振幅略有衰减但周期性清晰的拉比振荡的判据是：拉比振荡频率≥5倍的退相干速率。原子体系电子激发态弛豫的寿命通常是10ns左右，请问上述条件下能看到原子的拉比振荡吗？ 3）如果激光功率降低到4mW，还能看到拉比振荡吗？

  4. 核自旋的退相干很慢，已知某 NMR两能级体系的退相干时间为1ms，跃迁偶极矩大小为1.5D

    1. 计算该体系的退相干速率\(\Gamma\)；
    2. 按照以上判据，求能观察到拉比振荡的最小拉比频率；
    3. 计算满足该拉比频率所需的最小光场电场强度\(E_0\)；
    4. 由于黑体辐射产生的是多频域宽谱，即使上面那么大的电场强度通过黑体辐射产生也不现实，实际核磁设备中都是通过射频线圈产生定向、单色的强射频场。请以600MHz核磁跃迁频率为例，计算如上对应的电场强度如果通过黑体辐射来产生，需要把黑体加热到多少度。

谱能量密度频率分布图 #### 绘图Python代码 ```python import numpy as np import matplotlib.pyplot as plt

基本常数

h = 6.626e-34 c = 3e8 kb = 1.3806e-23

普朗克公式定义

def planck_rho(nu, T): numerator = 8 * np.pi * h * (nu ** 3) / (c ** 3) exponent = h * nu / (kb * T) denominator = np.exp(exponent) - 1 denominator[denominator == 0] = 1e-20 # 避免除零 return numerator / denominator

黑体温度设置

T_sun = 6000 # 太阳 T_earth = 300 # 地球 T_cmb = 2.7 # 宇宙微波背景

频率范围：100MHz ~ 10PHz，覆盖三个黑体的峰值

nu = np.logspace(8, 16, 1000)

计算各温度的谱能量密度

rho_sun = planck_rho(nu, T_sun) rho_earth = planck_rho(nu, T_earth) rho_cmb = planck_rho(nu, T_cmb)

绘图

plt.figure(figsize=(10,6)) plt.loglog(nu, rho_sun, label=‘6000K 太阳’, color=‘orange’, linewidth=2) plt.loglog(nu, rho_earth, label=‘300K 地球’, color=‘blue’, linewidth=2) plt.loglog(nu, rho_cmb, label=‘2.7K 宇宙背景’, color=‘black’, linewidth=2)

标注峰值位置

nu_max_sun = 5.8789e10 * T_sun nu_max_earth = 5.8789e10 * T_earth nu_max_cmb = 5.8789e10 * T_cmb plt.axvline(nu_max_sun, color=‘orange’, linestyle=‘–’, alpha=0.7) plt.axvline(nu_max_earth, color=‘blue’, linestyle=‘–’, alpha=0.7) plt.axvline(nu_max_cmb, color=‘black’, linestyle=‘–’, alpha=0.7)

plt.xlabel(‘频率 ν (Hz)’, fontsize=12) plt.ylabel(‘谱能量密度 ρ(ν,T) (J·m⁻³·Hz⁻¹)’, fontsize=12) plt.title(‘黑体谱能量密度频率分布图’, fontsize=14) plt.legend(fontsize=12) plt.grid(True, which=“both”, ls=“–”, alpha=0.5) plt.xlim(1e8, 1e16) plt.show()
